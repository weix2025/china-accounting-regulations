# 中国会计和金融法规全集

本仓库收集了中国会计和金融相关的法规文件，适用于财务软件开发（对标用友T+、金蝶KIS）。

**270+ 个文件**，涵盖：
- 企业会计准则（基本准则 + 42个具体准则）
- 小企业会计准则
- 行业会计制度

所有文件均来自官方渠道（财政部官网、国家税务总局等），以Markdown或PDF格式保存，便于阅读和检索。
