# 2024-2025年最新会计法规汇总

## 法规清单

### 一、2024年发布的法规

1. **企业会计准则解释第18号**（财会〔2024〕24号）
   - 发布日期：2024年12月6日
   - 施行日期：2026年1月1日（允许提前执行）
   - 主要内容：浮动收费法下投资性房地产后续计量、保证类质量保证会计处理

2. **会计信息化工作规范**（财会〔2024〕11号）
   - 发布日期：2024年7月26日
   - 施行日期：2025年1月1日
   - 主要内容：规范数字经济环境下的会计工作，推动会计信息化健康发展

3. **会计软件基本功能和服务规范**（财会〔2024〕12号）
   - 发布日期：2024年7月29日
   - 施行日期：2025年1月1日
   - 主要内容：规范会计软件基本功能和服务，提高会计软件和相关服务质量

4. **中国注册会计师独立性准则第1号**（财会〔2024〕29号）
   - 发布日期：2024年12月31日
   - 施行日期：2025年7月1日
   - 主要内容：规范注册会计师在财务报表审计和审阅业务中的独立性要求

5. **关于严格执行企业会计准则切实做好企业2024年年报工作的通知**（财会〔2024〕26号）
   - 发布日期：2024年12月
   - 主要内容：指导企业规范执行会计准则，提升2024年年报质量

### 二、2025年发布的法规

1. **企业会计准则解释第19号**（财会〔2025〕32号）
   - 发布日期：2024年12月5日
   - 施行日期：2026年1月1日
   - 主要内容：非同一控制下企业合并中补偿性资产会计处理、处置同一控制下企业合并取得子公司时资本公积处理、电子支付系统结算金融负债终止确认、金融资产合同现金流量特征评估等

2. **关于严格执行企业会计准则切实做好企业2025年年报工作的通知**（财会〔2025〕33号）
   - 发布日期：2025年12月24日
   - 主要内容：指导企业规范执行会计准则，提升2025年年报质量

## 重点内容摘要

### 会计信息化相关
- 《会计信息化工作规范》和《会计软件基本功能和服务规范》自2025年1月1日起施行
- 首次将会计信息化写入会计法
- 规范数字经济环境下的会计工作
- 推动电子凭证会计数据标准应用

### 企业会计准则解释
- 解释第18号主要涉及投资性房地产和保证类质量保证的会计处理
- 解释第19号主要涉及企业合并、金融工具等领域的会计处理

### 年报工作要求
- 2024年和2025年年报工作通知分别对当年年报编制提出具体要求
- 重点关注长期股权投资、固定资产、无形资产、数据资源、收入、金融工具等领域
