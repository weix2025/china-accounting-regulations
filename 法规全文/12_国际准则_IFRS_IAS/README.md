## 重要准则说明

### 金融工具相关准则
- **IFRS 7** - 金融工具披露：规定金融工具的风险披露要求
- **IFRS 9** - 金融工具：取代IAS 39，规定金融资产的分类、计量和减值
- **IAS 32** - 金融工具列报：规定金融工具的列报要求
- **IAS 39** - 金融工具确认和计量：已被IFRS 9取代，但仍具参考价值

### 收入与租赁准则
- **IFRS 15** - 客户合同收入：统一收入确认模型（待下载）
- **IFRS 16** - 租赁：承租人单一租赁会计模型

### 企业合并与合并报表
- **IFRS 3** - 企业合并：规范企业合并的会计处理
- **IFRS 10** - 合并财务报表：建立单一控制模型（待下载）
- **IFRS 11** - 合营安排：规范合营安排的会计处理
- **IFRS 12** - 在其他主体中权益的披露：统一披露要求

### 资产相关准则
- **IAS 16** - 不动产、厂场和设备
- **IAS 36** - 资产减值
- **IAS 38** - 无形资产
- **IAS 40** - 投资性房地产
